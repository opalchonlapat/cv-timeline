# A timeline CV

A timeline visualization of your experience that you put in your CV.

## Getting Started

### Prerequisites
* Python 3.5+
* Virtualenv
```sh
pip install virtualenv==16.7.9
```

### Installation

1. Install all requirements
```sh
make install
```
